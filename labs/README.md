# Networking Labs Index

## Current Labs

### [Lab 1: TCP/IP Fundamentals](01-tcp-ip-fundamentals.md)
**Duration:** 30-45 minutes  
**Difficulty:** Beginner  
**Topics:** Network stack, TCP vs UDP, packet analysis  
**Tools:** ipconfig, netstat, telnet, tracert

### [Lab 2: DNS Troubleshooting](02-dns-troubleshooting.md)
**Duration:** 45-60 minutes  
**Difficulty:** Intermediate  
**Topics:** DNS resolution, cache management, server testing  
**Tools:** nslookup, ipconfig, DNS servers

### [Lab 3: IP Addressing & Subnetting](03-ip-addressing.md)
**Duration:** 60-90 minutes  
**Difficulty:** Intermediate  
**Topics:** Subnetting, network calculations, ARP  
**Tools:** ipconfig, ping, arp, route

### [Lab 4: Local Network Diagnostics](04-local-network-diagnostics.md)
**Duration:** 45-75 minutes  
**Difficulty:** Advanced  
**Topics:** Network troubleshooting, performance analysis  
**Tools:** netstat, pathping, netsh, wmic

## Prerequisites
- Windows 10/11 with command prompt access
- Basic networking knowledge
- Administrative privileges (recommended)
- Active internet connection

## Lab Environment
All labs use Windows built-in tools and are designed for:
- Home/office networks
- Non-production environments
- Educational purposes only
